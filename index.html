CI-Generated Data Viewer

About This Application

This single-page web application fetches and displays data generated by a GitHub Actions CI pipeline. The CI pipeline processes an Excel file (data.xlsx) using a Python script (execute.py), linter checks with ruff, and then outputs the processed data as result.json. This result.json is then published to GitHub Pages, and this page retrieves and shows it.
